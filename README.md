# mktpl
Make Template is a small overcomplication of a copy/paste program used for making templates.  

It lets you create templates inside ~/.templates and create them anywhere else.  

## How to install
To install you'll need `luarocks` and the following packages:  
- argpase: >= 0.7.1-1  
- luafilesystem >= 1.8.0-1  
- lunajson >= 1.2.3-1  

Do I specifically need a version at or greater than whats listed?  
No, probably not. These are the versions I had installed when creating the program.
